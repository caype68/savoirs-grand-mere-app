# Savoirs de Grand-Mère — UI Kit complet

Contenu :
- 16 icônes SVG
- 8 textures PNG
- 10 illustrations/covers PNG
- 16 composants PNG
- 4 écrans mockup PNG
- 1 palette + design tokens JSON/CSS
- 1 fichier manifest.json

Style :
- Thème sombre moderne
- Sobre, lisible, chaleureux
- Rappels graphiques “carnet / herboristerie / remèdes de grand-mère”
- Prévu pour mobile app iOS / Android

Conseil d’intégration :
- Utiliser `docs/tokens.css` ou `docs/tokens.json`
- Les assets SVG peuvent être recolorisés dans React Native / Flutter / web
- Les textures sont discrètes pour rester compatibles avec une UI moderne

Arborescence :
- colors/
- icons/
- textures/
- illustrations/
- components/
- screens/
- docs/
